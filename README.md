# Quiz


## Getting Started

### Prerequisites

- Node.js 
- npm 
- Browser

### Installation

1. Clone the repository:
    "git clone https://github.com/phyothiha-aung/quiz-rt.git"

2. Go to Root directory

3. Install depencies by typing 
    "npm install"

4. Start development environment: 
    "npm run dev"

5. Press "o" to open browser
